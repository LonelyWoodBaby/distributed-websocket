# distributed-websocket
> 分布式推送项目，利用rocketMq+netty实现分布式推送

### 技术框架：

| 框架名称    | 版本号 |
| ----------- | :----: |
| Spring Boot | 2.1.4  |
| Rocket-MQ   | 4.3.1  |
| Netty       | 4.1.34 |
| JDK         |   11   |
| Jackson     | 2.9.8  |

### 项目说明

- Rocketmq-starter：实现RocketMq的基本开发
- websocket-starter：实现基于Netty的推送管理模块
- xxxx-example：包含服务端与客户端的实例

### 使用说明：

详见websocket-client-example和websocket-server-example实例项目
